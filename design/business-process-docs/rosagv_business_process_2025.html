🏭 房間內眼鏡製程流程

🎯 房間內製程目標

專注於房間內的眼鏡鏡片精密處理工序，包括清洗、化學處理、預烘、烘烤等關鍵製程，確保產品品質達到最高標準

📦
料件接收

Loader AGV 從傳送箱接收待處理的鏡片

🚛 負責 AGV: Loader AGV

📋 Work ID: 2001-2004: 取入口傳送箱，支援 4 個接收端口

🧽
清洗處理

Loader AGV 執行清洗機的放取作業

🚛 負責 AGV: Loader AGV

📋 Work ID: 2011: 放清洗機, 2021: 取清洗機

🧪
泡藥處理

化學處理提升鏡片表面特性

🚛 負責 AGV: Loader AGV

📋 Work ID: 2031-2036: 放泡藥機, 2041-2046: 取泡藥機，支援 6 個泡藥槽

🌡️
預烘處理

預備烘乾程序，為後續烘烤做準備

🚛 負責 AGV: Loader AGV → Unloader AGV

📋 Work ID: 2051-2058: Loader 放預烘, 3001-3004: Unloader 取預烘，支援 8 個預烘位

🔥
烘烤處理

高溫烘烤程序，完成鏡片最終處理

🚛 負責 AGV: Unloader AGV

📋 Work ID: 3011-3012: 放烘烤機, 3021-3022: 取烘烤機，高溫處理工站

📤
料件輸出

Unloader AGV 將處理完成的鏡片送至出口

🚛 負責 AGV: Unloader AGV

📋 Work ID: 3031-3032: 放出口傳送箱，完成房間內製程循環，準備交接至房間外流程

2011, 2021: 清洗機放取作業

2051-2058: 放預烘機

Unloader AGV - 卸載車

房間內後段製程專家

  • 房間內製程完成點

🔄 房間內 AGV 協作工作流程

📦 Loader AGV - 料件接收
執行 Work ID 2001-2004，從入口傳送箱接收來自房間外的鏡片，開始房間內製程處理。
🧽 Loader AGV - 清洗處理
Work ID 2011 放入清洗機，經過清洗處理後，Work ID 2021 取出清洗完成的鏡片。
🧪 Loader AGV - 泡藥處理
Work ID 2031-2036 將鏡片放入6個泡藥槽進行化學處理，完成後 Work ID 2041-2046 取出。
🔄 AGV 協作切換 - 預烘工站
Loader AGV 執行 Work ID 2051-2058 放入預烘，Unloader AGV 執行 Work ID 3001-3004 從預烘取出，實現協作銜接。
🔥 Unloader AGV - 烘烤處理
Work ID 3011-3012 放入烘烤機，Work ID 3021-3022 取出完成高溫處理的鏡片。
📤 Unloader AGV - 料件輸出
Work ID 3031-3032 放入出口傳送箱，完成房間內所有製程，準備交接至房間外流程。

🎯 房間內協作重點

Loader ↔ Unloader 接力點：預烘工站是房間內兩種 AGV 的主要協作點。Loader AGV 負責前段製程（料件接收→清洗→泡藥→放預烘），Unloader AGV 負責後段製程（取預烘→烘烤→料件輸出），實現房間內製程的完整銜接

⏱️ 房間內典型作業時序流程

📋 房間內任務啟動 (T+0分)
房間內製程任務啟動，Loader AGV 準備接收來自房間外流程的鏡片，開始房間內精密製程處理。
🏗️ Loader AGV 料件接收 (T+1分)
Loader AGV 執行 Work ID 2001-2004，從入口傳送箱接收來自房間外的鏡片，開始房間內的精密製程處理流程。
🧽 清洗處理 (T+3分)
Loader AGV 執行房間內首道製程，Work ID 2011 放入清洗機，經過精密清洗處理後，Work ID 2021 取出清洗完成的鏡片。
🧪 泡藥處理 (T+8分)
進入房間內核心製程，Work ID 2031-2036 將鏡片放入6個泡藥槽進行化學處理，完成後 Work ID 2041-2046 取出處理完成的鏡片。
🔄 房間內 AGV 協作切換 (T+20分)
Loader AGV 執行 Work ID 2051-2058 將鏡片放入預烘工站，完成前段製程。Unloader AGV 準備接手後段製程，實現房間內 AGV 協作切換。
🌡️ 預烘和烘烤 (T+25分)
Unloader AGV 接手房間內後段製程，Work ID 3001-3004 從預烘取出，Work ID 3011-3012 放入烘烤機，完成房間內最終高溫處理工序。
📤 房間內製程完成 (T+35分)
Unloader AGV 執行 Work ID 3021-3022 從烘烤機取出完成品，Work ID 3031-3032 放入出口傳送箱，完成房間內所有製程，準備交接至房間外流程。
✅ 房間內任務完成 (T+40分)
房間內製程循環完成，AGVCUI 記錄房間內製程數據，OPUI 顯示房間內任務完成狀態，系統準備處理下一批次或交接至房間外流程。
