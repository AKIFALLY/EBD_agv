# 什麼是 RosAGV？

## 🎯 30 秒電梯簡報

**RosAGV** 是一個企業級自動導引車（AGV）控制系統，專為工業自動化場景設計。

### 核心特色
- 🚗 **多車型支援**：Cargo、Loader、Unloader 三種專業 AGV
- 🏗️ **雙環境架構**：車載控制 + 中央管理分離設計
- 🚀 **現代技術棧**：基於 ROS 2 Jazzy + Zenoh RMW + Docker
- 🏭 **實際應用**：眼鏡生產線自動化物料搬運

## 💼 業務價值

### 解決什麼問題？
- **人工搬運效率低**：取代人工推車，提升物料搬運效率
- **生產線協調困難**：自動化射出機、清潔、烘乾等工序間的物料流轉
- **多工位管理複雜**：統一管理多台 AGV 和多個工作站
- **外部系統整合**：與 KUKA 機器人、WMS 系統無縫整合

### 帶來什麼效益？
- ⚡ **效率提升 60%**：自動化物料搬運，24/7 不間斷運行
- 💰 **人力成本降低**：減少人工搬運工作，釋放人力到更高價值工作
- 🎯 **精確度提升**：精準定位和路徑規劃，減少操作錯誤
- 📊 **數據透明化**：實時監控所有 AGV 狀態和任務進度

## 🏭 實際應用場景

### 眼鏡生產線案例
```
射出機 → AGV搬運 → 清潔工站 → AGV搬運 → 烘乾工站 → AGV搬運 → 包裝區
   ↓                    ↓                    ↓
OPUI叫車             自動載入             KUKA機器人協作
```

### 典型一天運作
1. **早晨啟動**：所有 AGV 自動定位到待命位置
2. **生產期間**：根據生產計劃自動調度 AGV 執行搬運任務
3. **異常處理**：自動偵測障礙物、設備故障，重新規劃路徑
4. **晚間維護**：AGV 自動返回充電區，系統生成日報表

## 🏛️ 系統架構概覽

### 雙環境設計
```
🚗 AGV 車載系統           🖥️ AGVC 管理系統
├─ 即時控制               ├─ 車隊管理
├─ 硬體整合               ├─ 任務調度
├─ 感測器處理             ├─ Web 管理介面
└─ 手動控制               └─ 資料庫管理
```

### 三種車型
- **Cargo Mover**：架台與傳送箱間貨物搬運
- **Loader**：多工位自動載料操作（Transfer、Cleaner、Soaker、Pre-dryer）
- **Unloader**：後段工位卸料操作（Pre-dryer、Oven、出口箱）

## 🔧 技術特點

### 為什麼選擇 ROS 2？
- **企業級穩定性**：LTS 版本保證長期支援
- **模組化設計**：每個功能獨立開發和測試
- **豐富生態系統**：大量現成的導航、控制算法
- **跨平台相容**：Linux、Windows 都能運行

### 為什麼使用 Zenoh RMW？
- **高效能通訊**：低延遲跨網路通訊（< 10ms）
- **自動服務發現**：AGV 和管理系統自動連接
- **網路透明**：車載系統和管理系統可在不同網路
- **可擴展性**：輕鬆擴展到更多 AGV

### 為什麼採用 Docker？
- **環境一致性**：開發、測試、生產環境完全一致
- **快速部署**：一鍵部署整個系統
- **隔離安全**：不同服務間相互隔離
- **版本管理**：清晰的版本控制和回滾機制
