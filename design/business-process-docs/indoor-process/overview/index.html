📊 房間內製程系統概覽

專注於房間內的眼鏡生產製程，涵蓋清洗、泡藥、預烘、烘烤等核心工序的 AGV 自動化操作

⚠️ 系統工具使用前提條件

使用 RosAGV r 工具集進行系統診斷和管理之前，必須將 RosAGV 目錄加入 PATH 環境變數

在 ~/.bashrc 中添加：export PATH="/home/ct/RosAGV:$PATH"

重新載入環境：source ~/.bashrc | 驗證配置：which r

房間內製程診斷：r agvc-check, r containers-status

🎯 房間內製程特色

房間內製程專注於精密的化學處理和溫度控制，是整個眼鏡生產流程中最關鍵的製程控制環節。系統採用 Loader AGV 和 Unloader AGV 協同作業，確保製程的連續性和一致性。

🏭
核心工站

4個關鍵工站

  • Transfer 傳送箱：料件接收和分發
  • Cleaner 清洗機：鏡片清潔處理
  • Soaker 浸潤機：化學溶液處理
  • Pre-dryer 預烘機：預乾燥處理
🤖
AGV 配置

雙 AGV 協同系統

  • Loader AGV：負責料件裝載和工站間搬運
  • Unloader AGV：負責完成品卸載和後段處理
  • 自動調度：自動任務分配和路徑最佳化
  • 狀態同步：即時狀態監控和回報
⚙️
技術架構

現代化技術棧

  • ROS 2 Jazzy：最新的機器人作業系統
  • Zenoh RMW：高效能通訊中間件
  • Docker 容器化：雙環境部署架構
  • AI WCS 決策引擎：自動任務調度
📊
效能指標

關鍵效能數據

  • 處理能力：每小時 120+ 批次
  • 精度控制：±0.5mm 定位精度
  • 可用率：99.5% 系統可用時間
  • 品質保證：100% 製程追蹤記錄
